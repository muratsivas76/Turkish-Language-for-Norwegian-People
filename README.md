# Turkish Language for Norwegian People

Welcome to the **Turkish Language for Norwegian People** repository!  
This project aims to provide free, open-source, and high-quality Turkish language learning materials specifically designed for Norwegian speakers.

## About This Project

- All materials are shared under the [GNU General Public License v3 (GPL v3)](https://www.gnu.org/licenses/gpl-3.0.en.html), ensuring freedom to use, modify, and distribute.
- The content includes text files, PDFs, videos, and audio recordings to support diverse learning styles.
- This project is inspired by the spirit of sharing, collaboration, and open knowledge.

## Repository Structure

- `txt/` - Text-based lessons and exercises
- `pdf/` - Printable lesson materials and booklets
- `movs/` - Video lessons and explanations
- `audios/` - Audio lessons and pronunciation guides
- `images/` - Image files 

## How to Use

- Browse the folders to find materials suitable for your learning needs.
- Feel free to download, share, and contribute improvements.
- Contributions are welcome! Please see [CONTRIBUTING.md](CONTRIBUTING.md) for guidelines.

## About the Author

This project is maintained by a graduate of Istanbul University, Faculty of Literature (Class of 2000), with years of experience teaching Turkish to non-native speakers.
